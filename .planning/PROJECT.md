# 跃鳞科技企业官网

## What This Is

这是深圳市跃鳞科技有限公司的企业形象官网，面向设备制造商、系统集成商和终端工厂采购客户。网站以品牌展示为主，兼顾询盘与留资，重点表达公司的工业传感器产品、解决方案能力、合作品牌生态与技术服务能力。

## Core Value

让潜在客户快速理解跃鳞科技卖什么、能解决什么问题、为什么值得联系，并顺畅发起咨询。

## Delivery Scope

### 已交付

- 中文与英文双语站点结构
- 浅色与深色两套主题
- 首页、产品中心、解决方案、服务与支持、关于我们、联系我们
- 五个一级产品分类，以及工业传感器的参考站子类结构
- 真实可提交的询盘表单、后端校验与数据库入库
- Docker Compose 一键启动前后端与 PostgreSQL

### 后续内容补充项

- 产品实拍图、型号清单、说明书与下载资料
- 品牌授权图、资质证书与项目交付证明
- 最终联系电话、邮箱与业务受理人
- 英文文案的上线前润色

## Constraints

- 首版必须支持中文与英文双语
- 技术栈固定为 React + Vite + React Query + React Router + VChart，以及 Python 3.11 + FastAPI + SQLAlchemy 2 + Alembic + PostgreSQL
- 部署方式固定为 Docker Compose
- 页面必须支持浅色与深色两套主题
- 官网以品牌展示为主，兼顾询盘与留资
- 产品中心采用五个一级分类，并在工业传感器下扩展参考站点细分子类
- 当前素材有限，首版允许结构化占位，但站点体验不能显得像未完成模板
- 页面风格采用 Precision Industrial，避免电商风格和过度营销化表达

## Context

- 公司名称：深圳市跃鳞科技有限公司
- 公司地址：深圳市南山区粤海街道科技园社区科苑路 8 号讯美科技广场 3 号楼
- 主要方向：工业传感器、安全防护传感器、激光测距传感器、线性滑轨及模组、气动元器件
- 解决方案：工业自动化解决方案、软件开发、技术集成
- 合作品牌：华怡丰 Huayifeng、松下 Panasonic、西克 Sick、亚德客 AirTAC、金器 Mindman
- 视觉基调：深海蓝、钢灰、冷白为主，少量琥珀橙用于 CTA 与关键信息强调

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| 将项目定位为企业官网而非电商站 | 用户核心目标是展示品牌与承接询盘，不是在线交易 | 已采纳 |
| 首版同时支持中文和英文 | 官网需要覆盖本地沟通与国际化展示场景 | 已采纳 |
| 产品中心使用五个一级分类 | 既符合用户业务表达，也便于后续扩展真实目录 | 已采纳 |
| 工业传感器子类优先参考华怡丰产品站 | 参考已有成熟目录结构，可以降低首版信息架构成本 | 已采纳 |
| 使用 Precision Industrial 作为 UI 方向 | 避免模板化科技站，更贴近工业技术企业气质 | 已采纳 |
| 询盘链路必须真实入库 | 联系页不应停留在 contract 占位级别 | 已采纳 |
| Compose 支持端口覆盖 | 交付环境不能因为本机端口占用而无法启动 | 已采纳 |
